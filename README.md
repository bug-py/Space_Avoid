 # Space Avoid 🚀
- ## Présentation
![Menu](/img/menu.png)
### **Space Avoid** est un jeu dans laquel on est une fusée 
![Jeu](/img/game.png)
### il faut esquiver des  astéroïdes ☄️ et survivre le plus longtemps possible ⏳
![Game Over](/img/over.png)

- ## Comment jouer ?
### Donc il faut installer la version de Python 3.12
### Et lancer cette commande (dans un env virtuel):
```bash
py -m pip install -r requirements.txt
```
- ## Logiciel Utilisé 

### Donc ce jeu est écrit en Python avec la  bibliothèque Pygame
### Les éléments graphiques sont du pixel art réalisé avec le logiciel [Piskel](https://www.piskelapp.com/) 
### Les sous ont été produits avec  [BeepBox](https://www.beepbox.co/) 
### *Les fichiers sources pour modifier les assets sont directement présents dans le projet*
